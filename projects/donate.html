Support Freedom Unchained

If you want to support Garrett and the broader Freedom Unchained mission financially, use the button below. The donation page you visit next may look generic, so this page explains how to confirm that you are on the correct donation path before you continue.

How to confirm it is the correct page

Before donating, check the web address on the page you are sent to.

  • The page should be on the Bridge Bucks Bank website.
  • The URL should include a_aid=Freedom_Unchained.
  • If that referral ID is visible in the address, you are on the correct donation path for Garrett / Freedom Unchained.

Important: The donation page itself may look similar to other pages in the same system. The key thing to verify is that the URL contains a_aid=Freedom_Unchained.

Before you continue

This donation handoff page exists to make the process clearer and easier to trust. It gives you a simple place to start and a simple way to verify that the donation path is connected correctly.
